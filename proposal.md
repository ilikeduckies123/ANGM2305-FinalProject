# Idle-Clicker Game

## Repository Link
https://github.com/ilikeduckies123/ANGM2305-FinalProject.git

## Description
A short idle clicker game. Will have a gameplay loop that can be played forever.

## Features
Click to gain money
Buy upgrades to make money over time
Utilize alternate forms of currency for player engagement
Achievements to fuel gameplay

## Challenges
Will need to research how to make an interactible window within Pygame
Research how to potentially save data

## Outcomes
### Ideal Outcome
Create an engaging player experience that feels unique. If given more time, it could expand into a fully realized game.

### Minimal Viable Outcome
Create a button the player can click to raise currency.
Create a button the player can click to gain currency over time.

## Milestones
### Week 1 
Setup Pygame window
Setup currency tracker
### Week 2 
Create a button for the player to interact with to raise currency
Create a button for the player to interact with to gain money over time
### Week 3
Create a loop for the buttons to raise values at specific amounts, so it can be played forever
### Week 4
Create achievement system
Create alternate currency to be used
Create new upgrades using currency system
