## Description
A short idle clicker game. Will have a gameplay loop that can be played forever.

## Features
### Click to gain money
#### Create currency that will increase when clicking a specified area
### Buy upgrades to make money over time
#### When clicking dedicated area, create a loop that raises money by a set amount every second
### Utilize alternate forms of currency for player engagement
#### Create "money 2." Essentially the same as money but different upgrades
### Achievements to fuel gameplay
#### Once players hit specific milestones play an effect to celebrate, potentially include achievement window.
